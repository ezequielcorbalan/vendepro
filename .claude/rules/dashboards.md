# Dashboard Rules

## Types
- General: executive overview (KPIs, funnel, trends)
- Per-area: operational detail (leads, appraisals, properties, sales)
- Per-agent: individual performance

## Must include
- KPI cards with clear labels and trend indicators
- Conversion funnel (lead → contact → appraisal → capture → publication → reservation → sale)
- Time-based evolution (monthly/weekly)
- Pending/overdue alerts
- Agent ranking/comparison (admin view)

## Design
- Desktop-first layout
- Charts: Recharts (BarChart, PieChart, LineChart)
- Custom SVG for funnels
- Responsive grids that collapse to 1-2 columns on mobile
