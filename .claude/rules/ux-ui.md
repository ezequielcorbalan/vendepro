# UX/UI Rules

- No empty/decorative screens — every page must be useful for work
- Cards must show operational info: owner, urgency, next step, status
- Kanban boards need filters, SLA alerts, and quick actions
- Calendars must be operational tools, not just visual
- Useful information density without losing clarity
- Consistent color coding across the app (event types, stages, status)
- Quick actions: call, WhatsApp, complete, reschedule — always accessible
- Modals for creation, inline for quick edits
- Brand colors: #ff007c (pink primary), #ff8017 (orange accent)
- Font: Poppins via Google Fonts CDN
