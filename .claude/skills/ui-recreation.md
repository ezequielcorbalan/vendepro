# Skill: UI Recreation from Screenshots

## Workflow
1. Receive screenshot or design reference
2. Implement the UI in code
3. Take screenshot of result
4. Compare side-by-side with original
5. List all mismatches (colors, spacing, alignment, content)
6. Fix mismatches
7. Repeat comparison (minimum 2 rounds)

## Rules
- Don't invent design — match the reference exactly
- Don't "improve" the design unless asked
- For CRM/dashboard screens, prioritize operational fidelity
- Match spacing, font sizes, colors, border radii
- Test both mobile and desktop
